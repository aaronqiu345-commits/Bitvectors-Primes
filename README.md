## Bit Vectors & Primes
How to run: \
Open terminal in file. \
Run command: make
./primes -n x (-s) (-p) \

-n x: Required. X determines the maximum number to count to. \
-s: Lists all prime numbers and interesting details if existent. \
-p: Lists prime numbers that are palindromic in bases 2, 9, 10, and 27. \
